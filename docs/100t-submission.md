## Project Intro (≤500 words)

**CodeBot** is a web-based AI-powered virtual mech builder that gamifies the developer experience. Developers create and evolve a pixel-art mech bot that grows stronger with coding activity, AI-powered challenges, and turn-based combat — all powered by Xiaomi MiMo V2.5 Pro.

### Problem
Developer engagement and continuous learning are hard to sustain. Traditional coding challenge platforms feel transactional — complete a task, get points, repeat. There's no emotional connection, no visual progression, and no social competition layer that keeps developers coming back.

### Solution
CodeBot turns coding activity into a living, evolving mech robot. Every action — completing AI challenges, chatting with the AI engineer, battling other developers, joining guilds — feeds into a persistent character that visually evolves through 5 stages: Frame → Chassis → Armor → Full Mech → Legend.

### AI Features (Powered by Xiaomi MiMo V2.5 Pro)

1. **AI Engineer Chat** — A conversational AI assistant built into the platform. Developers chat with their mech's AI engineer for coding advice, debugging help, and personalized training recommendations. Uses streaming responses for real-time interaction.

2. **Daily AI Challenges** — Fresh coding challenges generated by MiMo every day, difficulty-scaled to the user's level. Auto-generated, persisted to database, and deduplicated per user per day.

3. **Bot Lore Generator** — MiMo creates dramatic sci-fi backstories for each developer's mech — origin story, defining battles, and prophesied destiny. Personalized narrative layer that deepens engagement.

4. **AI Battle Commentary** — Real-time AI-generated dramatic narration for mech duels, making each fight feel like an epic story.

5. **MiMo Token Usage Dashboard** — Transparent public display of daily/monthly AI token consumption, showing real MiMo API usage metrics on the landing page.

6. **Code Review (planned)** — AI-powered code review integration for GitHub repos.

### Technical Approach

- **Framework:** Next.js 16 (App Router) with TypeScript
- **Database:** PostgreSQL via Prisma 6 ORM (Neon serverless)
- **AI Integration:** Xiaomi MiMo V2.5 Pro via OpenAI-compatible API, with SSE streaming and `reasoning_content` field support (MiMo-specific chain-of-thought)
- **Auth:** Session-based with bcrypt hashing
- **Deployment:** Vercel (edge-optimized)
- **Visual Design:** Pixel art aesthetic with self-hosted Press Start 2P + JetBrains Mono fonts

### Innovation
- AI isn't just a feature — it's the core engine that drives challenges, narratives, combat, and user engagement
- MiMo's reasoning capability (`reasoning_content` field) is used for transparent AI decision-making in challenge generation
- Gamification loop: Code → Earn XP → Evolve Mech → Unlock Features → Battle → Join Guilds → More Code

### Social Value
- Encourages continuous learning and coding practice through gamification
- Guild system fosters developer community and collaboration
- Public token usage dashboard promotes transparency in AI resource consumption
- Open-source project that other developers can learn from and contribute to
